## Latar Belakang Tugas

Sebagai tugas akhir dari masa kontrak sebagai *intern Data Scientist* di **ID/X Partners**, kali ini aku akan dilibatkan dalam projek dari sebuah *lending company*. Aku akan berkolaborasi dengan berbagai departemen lain dalam projek ini untuk menyediakan solusi teknologi bagi *company* tersebut. Aku diminta untuk membangun model yang dapat memprediksi *credit risk* menggunakan *dataset* yang disediakan oleh *company* yang terdiri dari data pinjaman yang diterima dan yang ditolak. Selain itu aku juga perlu mempersiapkan media visual untuk mempresentasikan solusi ke klien. Memastikan media visual yang aku buat jelas, mudah dibaca dan komunikatif. Pengerjaan *end-to-end solution* ini dapat dilakukan di *Programming Language* pilihanku dengan tetap mengacu kepada *framework/methodology Data Science*.
